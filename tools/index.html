    工具 - tools

    以下是我当前常用的软件、硬件、插件和一系列工具网站的列表清单，不定期更新。

    硬件

    • 工作笔记本电脑 - ThinkPad T14 Gen3
    • 鼠标 - 小米无线鼠标

    配件

    • 电脑支架 - 铝合金折叠笔记本支架

    软件

    • 博客（本网站） - 托管在 github 上
    • 浏览器 - Google Chrome
    • 编程 - 我一般较多使用 VSCode 。
    • 笔记软件 - 我试过很多款笔记软件，最终选择了 OneNote。
    • 屏幕截图 - Snipaste
    • 博文编辑markdown软件 - Typora

    移动应用程序

    • 微信读书- 简洁好用的读书软件
    • 不背单词、轻听英语 - 学英语的好帮手
    • 哔哩哔哩、小红书 - 常逛的综合资讯平台

    插件

    OneNote 插件

    • OneMore - 感谢这位开发者让我能够简单的使用目录、双链功能，我非常喜欢他提供的各类文本框。

    网站

    • Anna’s Archive - 目前最全的免费好用电子书网站。
    • 易书计划 - 跟踪免费电子书网站的最新消息。
    • 电子工业出版社悦学多媒体课程资源平台 - 免费电子教辅书。
    • 思谋学术 - 实时刷新谷歌学术的镜像网站。
    • 谷歌字体 - 数量有限的高质量字体。
    • material3 - 谷歌的设计系统。
    • fluent2 - 微软的设计系统。
    • materialUI - 提供配色参考，可以选择心仪的颜色，点击copy色值。
    • docuseal - PDF电子签名解决方案。
    • Excalidraw - 手绘风格的关系图绘制网站，方便随时梳理思路。
    • Font Awesome - 优质图标网站。
    • Writing Concisely - 学术写作指导。
    • CS自学指南 - 北大学长制作的CS自学指南，介绍了众多计算机领域权威课程，让我最喜欢的就是它的篇幅克制，一个学科只推荐两三门课。
