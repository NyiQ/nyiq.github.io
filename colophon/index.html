Trouble Shooting

博客部署过程中遇到的困难分为两方面：hexo设置 与 NEXT主题设置。此处将我遇到的问题以及相应解决方法记录如下：

hexo设置

  • 远程仓库怎么删除？

    由于我两年前设置过hexo，所以以往的记录都还保存在repo仓库中，想要清空再开始，参考的教程如下：《git清空远程仓库》《Git删除仓库中的文件和文件夹》

  • 误点了unpublish site按钮怎么办？

    在设置github仓库settings下的pages页面中，不小心点击了unpublish site，结果是选择github actions，然后在仓库内新建任意文件就可以重新打开网址。

  • Recv failure: Connection was reset报错怎么处理？

    参考教程：《git:上传代码时，出现fatal: unable to access ‘XXX‘: Recv failure: Connection was reset 错误解决方法（保姆级教学）》

  • github连接不上怎么办？

    参考教程：《如何ping通github》

NEXT设置

  • 首页文章如何截断，设置阅读全文？

    在文章内你想要截断的部分写入命令：<!--more-->。



网站结构

我的网站结构设置主要受到了 Mensur Duraković 和 Maggie 的启发，除了基本的首页Browse和个人页About以外，又额外增加了7个页面。虽然遵循了‘7+-2’原则，菜单项仍是略显冗余，但每个页面的设置都是有意义的，我并不打算删改。随着对NEXT主题的探索，这些菜单项将获得更好的组织呈现。下面是所有页面概述：

  • About - 个人简介。

  • Browse - 浏览主要文章。原本是home页，但由于我写的文章可以在此页面浏览查看，并没有承担真正的home的功能，所以更名成browse。

  • Categories - 文章分类总览。此页面是为了弥补Browse页面只能顺序观看的不足，满足总览需求。

  • Colophon - 建站分享。由于我的网站是通过较为常见的方式搭建的，所以网上能搜到许多帮助，我会将建站的经过分享在此页面，希望能帮助到更多的人。

  • Current - 个人动态。以月为单位分享一些个人阅读、探索和想法的进展、现状。

  • Library - 好书记录。我认为比较有用的或者个人喜欢的书籍。

  • Portfolio - 作品集。主要展示平面设计类作品。

  • Tools - 工具箱。记录常用的软件、硬件、插件及网站工具。

  • Websites - 资讯。尽管书籍还是获取高广度信息的重要来源，但已经不是唯一渠道了，现在有许多从书上无法轻易获得的资讯，我将一些自己常用的网站整理于这个页面。



最后，我赞成maggie的想法——文章就像是植物在生长这一概念，所以我的每一个帖子都会被不断完善，他们并不总会是刚发出来就非常完整的文章。但是想法会有成熟的一刻，我通过百分比来标记每个帖子的成熟度（但愿没有早夭的帖子）。
